# linear-models-tsa
Codes used for the Linear Models and Time Series Analysis Lecture Notes
These notes are used for the following courses:
* the undergraduate class __MATH 62.2 Time Series and Forecasting__
* the graduate class __MATH 271.2 Advanced TIme Series and Forecasting__
The following codes are included in this reposity:
* `2Regression.R` : codes for implementing multiple linear regression analysis
* `2ANOVA.R` : codes for implementing analysis-of-variance (including the dataset calcium.csv)
* `3TSA.R` : codes for implementing basic time series modelling (AR, MA)
* `3LinearModels.R` : codes for implementing ARIMA models for the mean equation
* `4VolatilityModels.R` : codes for implementing GARCH models for the volatility equation
* `5MTS.R` : codes for implementing VARMA models for multivariate time series analysis
